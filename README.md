# Angry Birds - Gesture Control Edition

愤怒的小鸟手势版 - 使用手势控制的网页版愤怒的小鸟游戏。

## 功能特点

- 🎮 **手势控制**：通过摄像头识别手势，捏合拉动发射小鸟
- 📱 **移动端支持**：支持触控拖拽操作，适配手机和平板
- 🖱️ **鼠标控制**：PC端可使用鼠标拖拽发射
- 🏗️ **5个关卡**：从简单到复杂的堡垒结构
- 🎯 **物理引擎**：真实的碰撞和堆叠效果
- 🔊 **音效反馈**：拉动和发射时的音效提示

## 操作说明

### 手势控制（需摄像头）
1. **允许摄像头访问**：游戏需要使用摄像头进行手势识别
2. **使用右手操作**：
   - 捏合拇指和食指开始瞄准
   - 保持捏合并拖动手势调整方向和力度
   - 瞄准至少2秒后松开手指发射
3. **目标**：消灭所有绿色小猪

### 触控/鼠标控制
1. **点击小鸟**：在小鸟附近点击开始拖拽
2. **拖动瞄准**：向右拖动增加力量，调整发射角度
3. **保持2秒**：瞄准时间需达到2秒
4. **松开发射**：释放后小鸟将向左飞出

## 技术栈

- **MediaPipe Hands**：手势识别
- **Canvas 2D**：游戏渲染
- **Web Audio API**：音效生成
- **Touch Events**：移动端触控支持

## 运行方式

### 方法1：直接打开
双击 `index.html` 文件在浏览器中打开（部分浏览器可能限制摄像头访问）。

### 方法2：本地服务器（推荐）

**Python:**
```bash
python3 -m http.server 8080
```

**Node.js:**
```bash
# 使用 npx
npx serve

# 或安装 http-server
npm install -g http-server
http-server
```

**VS Code:**
安装 Live Server 扩展，右键 `index.html` 选择 "Open with Live Server"。

然后访问 `http://localhost:8080`

> 注意：需要使用支持 WebRTC 的现代浏览器（Chrome、Firefox、Edge、Safari等）
